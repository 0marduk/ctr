# ct-spa

```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Runs the building of the Vue.js App (builds to dist/)
```
npm run dev
```

### Start the node server (localhost:8000 for dev)
```
node server.js
```
